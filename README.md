# personal_projects
This repo includes many personal learning projects that I have tried. These are a various levels of incomplete, but the intention was to learn and make something.

## Table of Contents

- learn_opengl
    - Gentle introduction into openGL via C++. The "learn_opengl" sub-folder is the first main chapter of the learn openGL book. The "learn_opengl_advanced" and "xml_parse_test" sub-folders are based on the second main chapter from the textbook, but also exploring creating a custom pipeline for defining objects in a scene (via an XML file), including loading the scene, generating objects, and then rendering the scene via vertex and fragment shaders.
- light_simulation_matlab
    - A basic 2D ray-tracing approach for simulating how photons of light reflect and refract through materials. This also includes a way to define objects and light sources in a scene, and simulating how rays travel through the scene.
- light_simulation_unity
    - Implementation of the "light_simulation_matlab" code to the Unity engine, allowing for a 3D ray tracing approach to simulating how light travels in a scene. The idea was (and still is!) to take this concept and expand this to create a fully fledged single player puzzle-based experience.
- book_of_shaders
    - Gentle introduction into fragment shaders using GLSL via [the book of shaders](thebookofshaders.com).
